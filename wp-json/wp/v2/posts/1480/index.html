I think it also solves part of the problem that [Chris Coyier](https:\/\/css-tricks.com\/full-bleed\/) was talking about when using the three-column grid approach in that the grid items need to be manually placed on the grid instead of it being naturally stacked on top of each other. <\/p>\n

>You have to apply the default behavior you want to every single element.
