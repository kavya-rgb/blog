\n What are attributes?
\n<\/h2>\n

In an easy way, it’s the first creepy thing you will learn in HTML that won’t make any sense at the time. Because, well it doesn’t affect much for most of them.<\/p>\n

But for some, it’s necessary like img<\/code>, a<\/code>, etc.<\/p>\n

In a professional way, as a web developer, we develop code for vast fo people and we need a beautiful and interactive website that a user will love. So we use them.<\/p>\n

I will talk about the common attributes,<\/p>\n

\n
\n <\/a>
\n src<\/code> and alt<\/code>
\n<\/h3>\n

src<\/code> is an essential attribute for the <audio>, <embed>, <iframe>, <img>, <input>, <script>, <source>, <track>, <video><\/code> tag to function. So in img<\/code> tag, we use something like this <img src=\"https:\/\/dev.to\/akhlakhossainjim\/my_image.png\" alt=\"\" ><\/code> So, src<\/code> use for mentioning the URL for the image. And don’t forget to use a descriptive alt<\/code> tag, because if anything goes wrong with the URL it will show the text instead. So finally It will look something like this <img src=\"https:\/\/dev.to\/akhlakhossainjim\/my_image.png\" alt=\"a demo image\" ><\/code>. Another reason for using alt<\/code> text is, the people uses a browser reader for their browsing experience the browser reader reads the alt<\/code> text for an img<\/code> tag.<\/p>\n

\n
\n <\/a>
\n href<\/code>
\n<\/h3>\n

It is necessary for the a<\/code> tag to work. It defines the address you will be going after clicking the text within an a<\/code> tag. So, we use it like <a href=\"https:\/\/ah-jim-seed.web.app\/\">my website<\/a><\/code>.<\/p>\n

\n
\n <\/a>
\n class and id
\n<\/h3>\n

Oh man, These are the most common day-to-day used attributes for you as a web developer. You will use them to style an element or make that interactive. You can use in any HTML element in the opening tag like <tagName class=\"\" id=\"\"><\/code> ie, <div class=\"demo__class\" id=\"unique_ID\"><\/code>. One thing to keep in mind here, the class<\/code> can be repeatable on the page but the id<\/code>s should be unique on that particular page and they are not required to write HTML code.<\/p>\n

\n
\n <\/a>
\n height, width, and style
\n<\/h3>\n

These are optional and you use them in any of the tags you can use in the body<\/code> tag. So, height defines the right of that element, width defines the width, and style let you use CSS within the tag itself. you can write them this way <p height=\"100px\" width=\"20px\" style=\"color: red;\">...<\/p><\/code><\/p>\n

\n
\n <\/a>
\n lang
\n<\/h3>\n

Lang is used to defining the language used in the element. The value are language code, it can be used as <html lang=\"en\"><\/code>.<\/p>\n
