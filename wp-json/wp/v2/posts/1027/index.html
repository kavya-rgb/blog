The rise of dark mode in recent years has made many website favicons feel awkward or even impossible to see in some cases. Provided you have the appropriate assets, it’s relatively easy to create a responsive favicon that can adapt to the user’s color scheme preferences.<\/p>\n

In order to create a responsive favicon, you need an SVG icon with as few colors as possible and two color palettes, one for light mode and one for dark mode. Usual rules about icon clarity and complexity apply, so make sure your icon meets all the necessary criteria to be visually distinguishable in any scenario. In our example, we will be using a monochrome icon from the fantastic Feather icon set<\/a>.<\/p>\n

Leveraging embedded styles in SVG images and the prefers-color-scheme<\/code> media query, we can create an appropriate <g><\/code> element to group all the elements of the icon. Then, using the id<\/code> of the group, we can apply the color palette for each design. Here’s what our final SVG asset looks like:<\/p>\n
