Please complete the following technical assignment. You can complete the project in any technology or language you choose however the solution must be suitable for a demo and be able to be reviewed at a source code level using standard text editing tools.<\/p>\n

classified email filter<\/p>\n

You have been hired by the White House to write software to detect potential classified mail being exchanged in non-secure email accounts. Your software will act as a filter to test the incoming email text, detect if the email might be classified, and additionally replace any sensitive text with censored \\*\\*\\*\\*\\* characters.<\/p>\n

Please write the core code as a function that accepts as parameters:<\/p>\n

\t\u2022\ta list of classified words\/phrases<\/p>\n

\t\u2022\tthe email text<\/p>\n

\t\u2022\ttrue\/false flag – If any of the classified words or phrases were located in the email<\/p>\n

\t\u2022\tcensored text \u2013 a new email text where the classified words or phrases have been replaced with asterisks, or the original email text if there was no classified material in the email <\/p>\n

Around the core function you can write a wrapper of your choice that lets a user input test data, run the function, and see the result.<\/p>\n
