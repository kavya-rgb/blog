\n htmx
\n<\/h3>\n

The philosophy of htmx is to write less Javascript code, this library gives you access to modern browser features such as AJAX, CSS Transitions, WebSockets and Server Sent Events directly in HTML, using attributes like hx-get<\/code> or hx-trigger<\/code>.<\/p>\n

A few considerable features of htmx:<\/p>\n

    \n
  • Any element, not just anchor tag or form can issue an HTTP request.<\/li>\n
  • Any event, not just form or button can trigger requests.<\/li>\n<\/ul>\n

    The quick demo below illustrate how to make POST request to create\/update data on a dynamic page.<\/p>\n
