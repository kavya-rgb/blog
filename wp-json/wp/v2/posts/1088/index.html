\n Overview of My Submission
\n<\/h3>\n

The application converts images to text and then to speech. The main target audience for this application would be visually impaired, who shall use the Braille technology inbuilt within iOS such that they can convert images to speech in realtime.<\/p>\n

\nGetting Started<\/h2>\n

The application uses Microsoft’s Azure cognitive services to convert text from images and then synthesise the text to speech. Microsoft Azure Translation Service is also added to the code base.<\/p>\n

\n Additional Resources \/ Info
\n<\/h3>\n

The application is made using SwiftUI, Azure Cognitive Services, Computer Vision, Firebase and Google Authentication. The app converts text to image, and then converts it to audio. A translate feature was also added to the code, however not implemented, due to time constraints. The translation was handled by Azure Cognitive Services.<\/p>\n
