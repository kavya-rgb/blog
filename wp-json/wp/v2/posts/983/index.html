\n What are Media Queries?
\n<\/h2>\n

Media Queries are chunks of CSS code that trigger when the rules provided to it are satisfied. <\/p>\n

These are the queries that help make the app or site responsive.<\/p>\n

The rules are called breakpoints.<\/p>\n
