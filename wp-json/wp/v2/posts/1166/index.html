\n Introduction
\n<\/h2>\n

<\/a>
React.js is a frontend JavaScript library for building amazing, sleek and performant user interfaces specifically for single page applications a.k.a<\/strong> SPAs<\/strong>. It is used for handling the view layer for web and mobile apps. React allows us to compose the entire user interface using the basic building blocks that are called as components. <\/p>\n
