Did you know there is a CSS property to adjust how numeric values are displayed?<\/p>\n

It’s called the font-variant-number<\/code> property, and it is used to distinguish how a particular font should render specific numeric values.<\/p>\n

Tailwind CSS has this support built-in so let’s look at the options we have and what they output.<\/p>\n

\n

Note: Not all fonts support this property, so be aware it might not work when you try this on custom fonts<\/p>\n<\/blockquote>\n

You can use the normal-nums<\/code> class to reset the value back to normal<\/code> whenever you want the default behavior.<\/p>\n
