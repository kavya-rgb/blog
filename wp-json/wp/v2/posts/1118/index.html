\n 1. Server Status Checker
\n<\/h3>\n

A server status checker is a script that will let you monitor if your server is active and running. It can show you how much downtime your server has had and give you regular updates of its performance. It’ll also send immediate alerts when server is down.<\/p>\n

Some of the tools that come in handy when developing such a script include:<\/p>\n

    \n
  • Socket – helps with connection to different servers.<\/li>\n
  • SSL – required for wrapping the socket connection.<\/li>\n
  • Date & Time – Check what time server is active or went down.<\/li>\n
  • smtplib – Send email alerts when the server is down.<\/li>\n
  • Pickle – Saves the history of the server performance.<\/li>\n<\/ul>\n
